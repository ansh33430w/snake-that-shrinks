# SNAKE THAT SHRINKS
A game made for HAck club ysws named twist

It is a classic retro game with a twist that your snakes shrinks when you you don't eat before timer A runs out your snake will shrink because of hunger
so in short it is game where your snake is super hungry and he need to eat very fast

but theres a catch theres A timer b and you should only eat after timer b runs out

like you have to time snake eating between timer A and timer B to get A sweet sopt


## Timer A 
it is a timer for snake shrink , if u don't eat within it your snake will shrink

## timer B
it's a timer for wait period after eating food u should wait till the timer runs out to eat


# features

1. background sound
2. food eating effect
3. game over effect
4. option menu where u can mute , change volume , change snakes color
5. shrink function
6. an score card
